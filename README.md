# AI-Powered Resume Analyzer

## Overview
An AI-powered web application that analyzes resumes against job descriptions and predicts ATS compatibility.

## Features
- ATS Score Generation
- Resume-Job Matching
- Skill Extraction
- Missing Skill Identification
- Resume Analysis Dashboard

## Technologies Used
- Python
- Flask
- HTML
- CSS
- PyPDF2
- Scikit-learn
- Matplotlib

## How to Run
1. Clone the repository
2. Install requirements
3. Run app.py
4. Open localhost in the browser
